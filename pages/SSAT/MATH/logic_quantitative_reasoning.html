Practice

  1. Find the next number: 2, 6, 18, 54, ___.
  2. Find the next number: 5, 9, 17, 33, ___.
  3. If all bloops are razzies and all razzies are lazzies, what must be true?
  4. In the pattern 3, 4, 7, 11, 18, ___, what comes next?
  5. A code changes CAT to DBU. What is DOG in the same code?
  6. If x is positive and y is negative, which must be positive: xy, x - y, y - x, or y/x?
Show Answer Key

1. 162. Multiply by 3.

2. 65. Add 4, 8, 16, then 32.

3. All bloops are lazzies.

4. 29. Each term is the sum of the previous two after 3 and 4.

5. EPH. Each letter moves forward one.

6. x - y, because subtracting a negative makes it larger.
